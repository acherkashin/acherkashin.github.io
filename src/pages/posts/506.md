Товарищи, подскажите, есть ли у кого опыт использования Code Connect в Figma? Стоит заморочиться и настроить?

Для тех, кто не знает: насколько я понимаю, мы можем настроить связь между компонентами в Figma и в коде. Тогда разработчику будет понятно, какой компонент использовать при реализации.

Но кто сейчас волнуется о разработчиках? Это поможет агентам при реализации дизайна. Сейчас, если вы попросите агента реализовать дизайн, сначала для стилей он будет использовать Tailwind, а затем уже как-то начнёт использовать ваши компоненты. Code Connect должен помочь сэкономить токены: агент сразу будет использовать именно ваши компоненты. Для этого нужно настроить маппинг между компонентами в Figma и в коде.
